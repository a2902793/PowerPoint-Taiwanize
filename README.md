# File-taiwanize

### 指令介紹
```bash
python taiwanized.py
```
![](https://user-images.githubusercontent.com/35146998/91635681-f0f0e080-ea2c-11ea-9b18-508eef70e152.png)

### 單一檔案翻譯
```bash
python taiwanized.py example.pptx
```


### 將資料夾內的所有 PowerPoint 檔案翻譯
```bash
python taiwanized.py -p
```
等效於 `--ppt` 或 `--pptx`
